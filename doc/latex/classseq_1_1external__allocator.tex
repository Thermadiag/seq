\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T, class External = detail\+::\+Malloc\+Free$>$\newline
class seq\+::external\+\_\+allocator$<$ T, External $>$}

Stl conforming allocator wrapper using an external class to perform the allocation. 


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & object type to allocate \\
\hline
{\em External} & class used to perform the actual allocation/deallocation.\\
\hline
\end{DoxyTemplParams}
\mbox{\hyperlink{classseq_1_1external__allocator}{external\+\_\+allocator}} is a stl conforming allocator that performs its allocation/deallocation based on the {\itshape External} class. External class must have the following signature\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{struct }external}
\DoxyCodeLine{\{}
\DoxyCodeLine{    \textcolor{keyword}{static} \textcolor{keywordtype}{void}* \mbox{\hyperlink{classseq_1_1external__allocator_a1fa79d4926dc641a842ddb857a120e29}{allocate}}(\textcolor{keywordtype}{size\_t} bytes) ;}
\DoxyCodeLine{    \textcolor{keyword}{static} \textcolor{keywordtype}{void} \mbox{\hyperlink{classseq_1_1external__allocator_a02da4fdefe81788b7fee07283a878d50}{deallocate}}(\textcolor{keywordtype}{void}* p, \textcolor{keywordtype}{size\_t} bytes);}
\DoxyCodeLine{\};}
\end{DoxyCode}


It is mainly used to test memory allocation libraries (like T\+C\+Malloc, jemalloc...) with stl like containers.
