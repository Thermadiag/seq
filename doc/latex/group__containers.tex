\doxysubsection{Detailed Description}
The \mbox{\hyperlink{group__containers}{containers}} module defines several container classes as alternatives to S\+TL containers or providing features not present in the S\+TL. These containers generally adhere to the properties of S\+TL containers, though there are often some associated A\+PI differences and/or implementation details which differ from the standard library. 

The Seq containers are not necessarly drop-\/in replacement for their S\+TL counterparts as they usually provide different iterator/reference statibility rules or different exception guarantees.
