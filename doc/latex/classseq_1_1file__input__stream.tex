\subsubsection*{template$<$unsigned buff\+\_\+size = 32$>$\newline
class seq\+::file\+\_\+input\+\_\+stream$<$ buff\+\_\+size $>$}

Input stream working on a F\+I\+LE pointer. 

\mbox{\hyperlink{classseq_1_1file__input__stream}{file\+\_\+input\+\_\+stream}} is a \mbox{\hyperlink{classseq_1_1basic__input__stream}{seq\+::basic\+\_\+input\+\_\+stream}} used to extract numbers, words and lines from a F\+I\+LE pointer or directly a filename. Its goal is to be able to use the optimized reading function (\mbox{\hyperlink{namespaceseq_a8dd14a20a2e7b03cebe3d48e3ae5e2ff}{seq\+::from\+\_\+stream}} to extract numerical values and words, \mbox{\hyperlink{namespaceseq_afe93dbc8ebf7707cd6a49a40efb69516}{seq\+::read\+\_\+line\+\_\+from\+\_\+stream}} to extract full lines) on a text file.

\mbox{\hyperlink{classseq_1_1file__input__stream}{file\+\_\+input\+\_\+stream}} uses internally a buffer of size \textquotesingle{}buff\+\_\+size\textquotesingle{} to read from the underlying F\+I\+LE object by chunks.
