\doxysubsection{Detailed Description}
Input stream working on a sequence of characters. 

\mbox{\hyperlink{classseq_1_1buffer__input__stream}{buffer\+\_\+input\+\_\+stream}} is a \#basic\+\_\+input\+\_\+stream used to extract numbers, words and lines from a sequence of characters. Use \mbox{\hyperlink{namespaceseq_a8dd14a20a2e7b03cebe3d48e3ae5e2ff}{seq\+::from\+\_\+stream}} to extract numerical values and words, \mbox{\hyperlink{namespaceseq_afe93dbc8ebf7707cd6a49a40efb69516}{seq\+::read\+\_\+line\+\_\+from\+\_\+stream}} to extract full lines.
