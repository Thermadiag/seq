\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T, class Allocator = std\+::allocator$<$\+T$>$, size\+\_\+t Align = Default\+Alignment$>$\newline
class seq\+::aligned\+\_\+allocator$<$ T, Allocator, Align $>$}

Allocator class with custom alignment. 


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & object type to allocate \\
\hline
{\em Allocator} & underlying allocator\\
\hline
\end{DoxyTemplParams}
\mbox{\hyperlink{classseq_1_1aligned__allocator}{aligned\+\_\+allocator}} is a stl conforming allocator class supporting custom alignment. It relies on an underlying allocator class to perform the actual memory allocation. \mbox{\hyperlink{classseq_1_1aligned__allocator}{aligned\+\_\+allocator}} will allocate more memory than required for over aligned types.

If {\itshape Align} is 0, the default system alignment is used.

If provided Allocator class is std\+::allocator$<$\+T$>$, \mbox{\hyperlink{classseq_1_1aligned__allocator}{aligned\+\_\+allocator}} is specialized to use \mbox{\hyperlink{namespaceseq_ac67ee9a9d6039de2b50c631129a0a1bc}{seq\+::aligned\+\_\+malloc}} and \mbox{\hyperlink{namespaceseq_a04be1ca38e79158feb4755807ef3d08c}{seq\+::aligned\+\_\+free}}.
