\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned buff\+\_\+size = 32$>$\newline
class seq\+::std\+\_\+input\+\_\+stream$<$ buff\+\_\+size $>$}

Input stream working on a std\+::istream. 

\mbox{\hyperlink{classseq_1_1std__input__stream}{std\+\_\+input\+\_\+stream}} is a \#basic\+\_\+input\+\_\+stream used to extract numbers, words and lines from a std\+::istream object. Its goal is to be able to use the optimized reading function (\mbox{\hyperlink{namespaceseq_a8dd14a20a2e7b03cebe3d48e3ae5e2ff}{seq\+::from\+\_\+stream}} to extract numerical values and words, \mbox{\hyperlink{namespaceseq_afe93dbc8ebf7707cd6a49a40efb69516}{seq\+::read\+\_\+line\+\_\+from\+\_\+stream}} to extract full lines) on a std\+::istream object.

\mbox{\hyperlink{classseq_1_1std__input__stream}{std\+\_\+input\+\_\+stream}} uses internally a buffer of size \textquotesingle{}buff\+\_\+size\textquotesingle{} to read from the underlying std\+::istream object by chunks. Even if the std\+::istream is internally bufferized, it makes the reading of single characters several times faster.
