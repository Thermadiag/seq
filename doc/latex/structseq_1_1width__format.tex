\doxysubsection{Detailed Description}
Class representing the width formatting for a any formatting object. 

\mbox{\hyperlink{structseq_1_1width__format}{width\+\_\+format}} controls how a string is aligned based on a width. The string can be aligned to the left, right or centered. If the given width is greater than the string length, the string is padded with a fill character. Otherwise, the string is truncated. Numerical values are never truncated.
