Detailed Description

template<class T, class Allocator = std::allocator<T>>
class seq::detail::chunk_pool_alloc< T, Allocator >

Allocate list_chunk objects (64 bytes aligned) using a seq::object_pool Note that the object_pool is only used after a few allocations to avoid a too big memory overhead for small sequences.
