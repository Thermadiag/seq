Detailed Description

template<unsigned buff_size = 32>
class seq::file_input_stream< buff_size >

Input stream working on a FILE pointer.

file_input_stream is a seq::basic_input_stream used to extract numbers, words and lines from a FILE pointer or directly a filename. Its goal is to be able to use the optimized reading function (seq::from_stream to extract numerical values and words, seq::read_line_from_stream to extract full lines) on a text file.

file_input_stream uses internally a buffer of size 'buff_size' to read from the underlying FILE object by chunks.
