Detailed Description

template<class T, class Allocator = std::allocator<T>>
class detail::HashBlockAlloc< T, Allocator >

Class used to allocate/deallocate T objects by block of 8, 16, 24, 32 or 64 elements. Internally uses a list of growing HashBlock objects. Uses the global GROW_FACTOR value
