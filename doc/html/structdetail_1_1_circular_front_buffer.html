Detailed Description

template<class T, class Allocator>
struct detail::CircularFrontBuffer< T, Allocator >

Circular buffer for the front buffer of a deque. Stores its elements in a consecutive way from max_size()-size to size. All insert operations consider that the buffer is not full.
