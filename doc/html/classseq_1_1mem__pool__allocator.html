Detailed Description

template<class T, class Allocator = std::allocator<T>, size_t Align = 0, class Lock = null_lock>
class seq::mem_pool_allocator< T, Allocator, Align, Lock >

Custom allocator based on mem_pool class.

mem_pool_allocator is a stl compliant allocator dedicated to node based containers like std::set, std::map, std::list, std::deque, std::multimap, std::multiset...

It provides faster allocation/deallocation time as well as reduced memory footprint (see mem_pool class for more details). Since the mem_pool class uses chunk based allocation, containers using mem_pool_allocator are usually more cache friendly, which fasten most operations (like iterating, sorting,...).

The Allocator template parameter is forwarded to the underlying mem_pool objects. It is also used when allocating more than one object. Therafore, using mem_pool_allocator does not benefit to containers relying on contiguous memory blocks like std::vector.
