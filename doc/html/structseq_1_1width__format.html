Detailed Description

Class representing the width formatting for a any formatting object.

width_format controls how a string is aligned based on a width. The string can be aligned to the left, right or centered. If the given width is greater than the string length, the string is padded with a fill character. Otherwise, the string is truncated. Numerical values are never truncated.
