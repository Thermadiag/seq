Detailed Description

template<class T, class Allocator = std::allocator<T>>
struct detail::deque_chunk_pool_alloc< T, Allocator >

Class used to allocate/deallocate list_chunk<T> objects. Internally uses a list of growing deque_chunk_pool objects. Uses the global GROW_FACTOR value
