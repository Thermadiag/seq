Detailed Description

template<class T, class Allocator = std::allocator<T>, size_t Align = DefaultAlignment>
class seq::aligned_allocator< T, Allocator, Align >

Allocator class with custom alignment.

Template Parameters
Tobject type to allocate
Allocatorunderlying allocator

aligned_allocator is a stl conforming allocator class supporting custom alignment. It relies on an underlying allocator class to perform the actual memory allocation. aligned_allocator will allocate more memory than required for over aligned types.

If Align is 0, the default system alignment is used.

If provided Allocator class is std::allocator<T>, aligned_allocator is specialized to use seq::aligned_malloc and seq::aligned_free.
