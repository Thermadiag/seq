Detailed Description

template<unsigned spin_count = 16U>
class seq::spin_mutex< spin_count >

Mutex-like class combining a spinlock and std::mutex.

spin_mutex is a kind of mutex that combines the speed of spin locking for uncontended scenarios and the CPU friendly wait of std::mutex.

On lock, spin_mutex first uses a spinning strategy for at most spin_count cycles. If the spinlock fails, then the std::mutex is locked.

Note that the std::mutex is only allocated when needed, i.e. at the first failed attempt to lock the spin_mutex with spinning. Therefore, the size of seq::spin_mutex is only 16 bytes on 64 bits systems.

spin_mutex follows the TimedMutex requirements.
