Detailed Description

template<unsigned buff_size = 32>
class seq::std_input_stream< buff_size >

Input stream working on a std::istream.

std_input_stream is a #basic_input_stream used to extract numbers, words and lines from a std::istream object. Its goal is to be able to use the optimized reading function (seq::from_stream to extract numerical values and words, seq::read_line_from_stream to extract full lines) on a std::istream object.

std_input_stream uses internally a buffer of size 'buff_size' to read from the underlying std::istream object by chunks. Even if the std::istream is internally bufferized, it makes the reading of single characters several times faster.
