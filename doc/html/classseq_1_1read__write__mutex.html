Detailed Description

template<class Mutex = spin_mutex<>>
class seq::read_write_mutex< Mutex >

A read-write mutex based on either seq::spinlock class or seq::spin_mutex class.

Template Parameters
Mutexeither seq::spinlock or seq::spin_mutex class

read_write_mutex follows the TimedMutex requirements.
