Detailed Description

template<class T, class Allocator = std::allocator<T>>
class detail::chunk_pool_alloc< T, Allocator >

Class used to allocate/deallocate list_chunk<T> objects. Internally uses a list of growing chunk_pool objects. Uses the global GROW_FACTOR value
