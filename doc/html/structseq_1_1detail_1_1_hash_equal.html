Detailed Description

template<class Hash, class Equal, bool EmptyHash = std::is_empty<Hash>::value, bool EmptyEqual = std::is_empty<Equal>::value>
struct seq::detail::HashEqual< Hash, Equal, EmptyHash, EmptyEqual >

@breif Gather hash class and equal_to class in the same struct. Inherits both for non empty classes. This is a simple way to handle statefull hash function or equality comparison function.
