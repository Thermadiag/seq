Detailed Description

template<class T, class External = detail::MallocFree>
class seq::external_allocator< T, External >

Stl conforming allocator wrapper using an external class to perform the allocation.

Template Parameters
Tobject type to allocate
Externalclass used to perform the actual allocation/deallocation.

external_allocator is a stl conforming allocator that performs its allocation/deallocation based on the External class. External class must have the following signature:

struct external
{
static void* allocate(size_t bytes) ;
static void deallocate(void* p, size_t bytes);
};

It is mainly used to test memory allocation libraries (like TCMalloc, jemalloc...) with stl like containers.
