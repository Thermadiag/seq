Detailed Description

template<class Key, class Value, class Hash, class Equal, class Allocator, LayoutManagement layout>
struct seq::detail::SparseFlatNodeHashTable< Key, Value, Hash, Equal, Allocator, layout >

Base class for robin-hood hash table. Values are stored in a seq::sequence object, and the hash table contains iterators to the sequence with additional hash value and distance to perfect location.
