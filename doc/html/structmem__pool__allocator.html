Detailed Description

template<class T, class Lock = null_lock>
struct mem_pool_allocator< T, Lock >

Custom allocator class based on mem_pool<T> class. Provides faster allocation deallocation as well as cache friendly object location.

Only usefull for allocation of one object at a time.

NOT working with std::list::sort() https://stackoverflow.com/questions/63716394/list-sort-fails-with-abort-when-list-is-created-with-stateful-allocator-when-com
